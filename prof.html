Les exercices

Le projet Exo7 propose aux étudiants des fiches d’exercices de mathématique avec indications et corrections de niveau L1-L2-L3. Nous proposons dorénavant des vidéos d'exercices et de cours de maths ! Ces documents sont élaborés, corrigés par des enseignants du supérieur. Le site propose en plus aux enseignants des exercices non corrigés.

Toutes les fiches d'exercices corrigées sont rendues publiques dans la partie destinée aux étudiants.

Vous trouverez le code source en saisissant le numéro de l'exercice dans le cadre de saisie.

Vous pouvez aussi récupérer tous les exercices (corrigés ou non) en un seul fichier (les exercices sont triés par thème, la numérotation absolue est entre crochets). Vous avez aussi à votre disposition les fichiers sources des polycopiés de cours (ainsi que les diapos beamer et les figures TikZ).

Questions fréquentes

  • J'ai trouvé une faute dans un exercice.
    Dans ce cas, merci de signaler l'erreur. N'oubliez pas de mentionner le numéro absolu que vous trouvez soit en consultant les exercices en ligne, soit dans le fichier LaTeX.

  • J'ai des problèmes avec les accents.
    Si vous obtenez du texte avec des symboles bizarres, changez l'encodage de votre fichier texte en utf8.

  • Où récupérer les images ?
    Dans le repertoire exo7.emath.fr/images/ ! Notez que pour certaines images vous avez auusi le fichier source (par exemple .fig pour xfig, ou .maple pour des commandes Maple,...).

  • Quelle est la licence d'utilisation ?
    La licence d'utilisation d'Exo7 se trouve à la page licence d'utilisation.
